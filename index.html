SME Mode

[]

SME Mode is a major mode for GNU Emacs which adds support for the SAP CPQ solution modeling language. The SME is a solution modeling environment provided by SAP SE. This mode supports SSC 2 and later.

This mode needs GNU Emacs 24. It will not work with GNU Emacs 23 and below. It has not been tested with other flavors of Emacs (e.g. Prelude).

Features

  1. Syntax highlighting
  2. Indentation and alignment of expressions and statements
  3. Tag navigation

Installation

Clone from Github: git clone https://github.com/mjj55409/sme-mode.git

Add your your .emacs.d/init.el file:

(add-to-list 'load-path (<your-path-to-sme-mode>))
(require 'sme-mode)

Usage

The major mode is enabled automatically for SME files with the extension '.ssc'.
